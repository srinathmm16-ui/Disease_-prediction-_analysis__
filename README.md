Disease Prediction Analysis Dashboard

Project Overview

This project presents a Disease Prediction Analysis Dashboard developed using Power BI. The dashboard helps analyze patient health data, identify disease patterns, evaluate risk categories, and support healthcare decision-making.

Tools Used

- Power BI
- Microsoft Excel
- Data Analysis

Dataset Information

- Total Patients: 200
- Attributes include Age, Gender, Smoking Status, Risk Score, Blood Pressure, and Predicted Disease.

Dashboard Features

- Disease Distribution Analysis
- Gender Distribution
- Smoking Status Analysis
- Risk Category Breakdown
- Average Risk Score Analysis

Key Insights

- Heart Disease was the most frequently predicted disease.
- Female patients represented 53% of the dataset.
- High-risk patients accounted for 40.5% of the total population.
- Smoking categories showed a nearly equal distribution.

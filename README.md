🏥 Disease Prediction Analysis Dashboard

📌 Project Overview

This project presents a Disease Prediction Analysis Dashboard developed using Power BI. The dashboard analyzes patient health data to identify disease patterns, evaluate risk levels, and support better healthcare decision-making.

---

🎯 Problem Statement

Healthcare providers often face challenges in identifying high-risk patients early due to large volumes of patient data. This project aims to simplify data interpretation and highlight critical risk factors.

---

🎯 Objective

- Analyze patient health data
- Identify key factors influencing disease prediction
- Build an interactive dashboard for easy visualization
- Support early detection and decision-making

---

🛠 Tools Used

- Power BI
- Microsoft Excel
- Data Analysis

---

📊 Dataset Information

- Total Patients: 200
- Attributes include:
  - Age
  - Gender
  - Smoking Status
  - Blood Pressure
  - Risk Score
  - Predicted Disease

---

📈 Key Insights

- Patients aged above 50 show a higher risk of disease
- Smoking significantly increases disease probability
- High blood pressure is strongly associated with higher risk scores
- Gender-based variations observed in disease distribution

---

📊 Dashboard Features

- Disease Distribution Analysis
- Gender-wise Analysis
- Risk Category Breakdown
- Interactive Filters for detailed insights
